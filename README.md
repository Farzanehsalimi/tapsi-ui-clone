# Tapsi Homepage Clone

A fully responsive recreation of the Tapsi landing page using **HTML** and **CSS**, built with clean structure and modern front-end practices.

---

## Features

- **One-page static layout**, fully hand-coded with semantic HTML
- **BEM methodology** for clean, scalable, and readable class naming
- **Responsive design** optimized for mobile, tablet, and desktop
- **Hero section with `<picture>`** to serve different background images per device size
- **CSS Root Variables (`:root`)** for consistent color theming
- Well-structured and reusable styles for:
  - Buttons
  - Badges
  - Titles and headings
  - Lists
  - Layout sections
- Optimized images to improve load performance

---

## Technologies Used

- **HTML5**
- **CSS3**
- **BEM (Block Element Modifier)**
- **Responsive design**
- **CSS variables (`var()`)**

---

## How to Run Locally

1. Clone the repository
2. Open `index.html` in your browser

Since this is a fully static project, no additional setup or build steps are required.

---

## Deployment

The project is deployed on **Vercel** and automatically served as a static site.
